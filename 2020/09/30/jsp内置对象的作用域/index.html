JSP 内置对象及其作用域

1. 几个常用的内置对象

JSP支持九个自动定义的变量，江湖人称隐含对象。这九个隐含对象的简介见下表：

对象 描述
request HttpServletRequest类的实例
response HttpServletResponse类的实例
out Print Writer类的实例，用于把结果输出至网页上
session HttpSession类的实例
application ServletContext类的实例，与应用上下文有关
config ServletConfig类的实例
pageContext PageContext类的实例，提供对JSP页面所有对象以及命名空间的访问
page 类似于Java类中的this关键字
exception exception 类的对象，代表发生错误的 JSP 页面中对应的异常对象

2.内置对象的作用域

  • request对象只在一次请求中有效,但是请求转发可以携带这个request对象.
  • session对象在一次会话中有效,用户在打开浏览器访问该网站后,将会分配到一个session对象,直到用户关闭浏览器,该session对象才会消失.
  • application对象在服务器中有效,生命周期为打开服务器到关闭服务器.
  • pageContext对象仅在当前页面有效
